# AI智能控制助手

一款基于HarmonyOS ArkTS开发的智能家居控制应用，支持设备管理、定时任务、AI控制等功能。

## 主要功能

### 🏠 设备管理
- **设备组展示**：以卡片形式展示所有设备组，显示设备组名称、位置、在线状态
- **设备单元控制**：进入设备组详情页，可控制控制型设备（开启/关闭），查看状态型设备数据
- **实时状态同步**：通过WebSocket实现设备状态实时更新
- **自定义图标**：支持为每个设备单元更换自定义图标，图标更换后持久化保存

### ⏰ 定时任务
- **任务管理**：创建、编辑、删除定时任务
- **多种定时类型**：
  - 每天：指定时间每天执行
  - 每周：指定周一到周日的组合执行
  - 一次性：指定日期和时间执行一次
- **任务启用/禁用**：通过开关控制任务状态
- **任务列表展示**：在设备单元卡片中显示关联的定时任务

### 🤖 AI智能控制
- **文字控制**：通过AI控制面板发送指令控制设备
- **批量控制**：支持切换设备组所有设备的状态

### ⚙️ 配置管理
- **服务器配置**：配置服务端IP地址、端口、用户名、密码
- **持久化存储**：登录信息和配置使用Preferences加密存储
- **页面配置**：通过下拉菜单快速打开配置对话框

### 📱 用户体验优化
- **响应式布局**：支持不同屏幕尺寸，自适应调整布局
- **错误处理**：完善的错误提示和异常处理机制
- **账户安全**：登录失败次数限制和账户锁定功能

## 技术架构

### 项目结构
```
entry/src/main/ets/
├── components/          # UI组件
│   ├── ScheduleEditDialog.ets    # 定时任务编辑对话框
│   └── ScheduleListComponent.ets # 定时任务列表组件
├── config/              # 配置文件
│   └── DeviceIcons.ets           # 设备图标配置（150+种图标）
├── model/               # 数据模型
│   ├── DeviceModel.ets           # 设备组和设备单元模型
│   ├── ScheduleModel.ets         # 定时任务模型
│   ├── ServerConfig.ets          # 服务器配置模型
│   └── IndexData.ets             # 页面数据模型
├── pages/               # 页面
│   ├── Index.ets                 # 主页面（设备组列表）
│   └── DeviceDetail.ets          # 设备详情页面
├── services/            # 服务层
│   ├── ApiService.ets            # HTTP API服务
│   ├── ScheduleService.ets       # 定时任务服务
│   └── WebSocketService.ets      # WebSocket实时通信服务
└── utils/               # 工具类
    ├── ApiCompatibilityUtil.ets  # API兼容性工具
    ├── DataSource.ets            # 数据源封装
    ├── IconStorage.ets           # 图标持久化存储（新增）
    ├── LoginStorage.ets          # 登录信息加密存储
    ├── ScreenAdaptUtil.ets       # 屏幕适配工具
    └── VoiceInputUtil.ets        # 语音输入工具（开发中）
```

### 核心技术
- **ArkTS**：HarmonyOS原生开发语言
- **ArkUI**：声明式UI框架
- **Preferences**：数据持久化存储
- **WebSocket**：实时状态同步
- **HTTP**：API通信

## 开发环境

- HarmonyOS SDK：API 9+
- DevEco Studio：4.0+
- Node.js：18+

## 快速开始

### 前置条件
1. 安装DevEco Studio 4.0+
2. 配置HarmonyOS SDK（API 9+）

### 打开项目
1. 启动DevEco Studio
2. 选择"Open Project"，打开项目根目录
3. 等待项目同步完成

### 构建项目
```bash
# 使用hvigorw构建HAP包
./hvigorw assembleHap

# 或在DevEco Studio中点击"Build" -> "Build Hap(s)/App(s)"
```

### 运行项目
1. 连接HarmonyOS设备或启动模拟器
2. 在DevEco Studio中点击运行按钮（绿色三角形）

## 使用说明

### 首次登录
1. 打开应用后，进入配置页面
2. 输入服务器地址、端口、用户名、密码
3. 点击"保存"按钮完成配置

### 设备控制
1. 在主页面点击设备组卡片进入详情页
2. 对于控制型设备，点击"开启"或"关闭"按钮控制设备
3. 对于状态型设备，查看实时状态数据

### 定时任务
1. 在设备详情页点击下拉菜单，选择"定时任务"
2. 点击"+"按钮创建新任务
3. 设置任务名称、执行时间、执行命令
4. 选择定时类型（每天/每周/一次性）
5. 点击"保存"完成创建

### 自定义图标
1. 在设备详情页点击下拉菜单，选择"页面配置"
2. 找到要更换图标的设备单元，点击"更换"按钮
3. 在图标选择界面选择新图标
4. 点击图标完成更换，图标会自动保存

### 修改配置
1. 在主页面点击左上角菜单按钮打开下拉菜单
2. 选择"配置"选项打开配置对话框
3. 修改服务器地址、端口、用户名或密码
4. 点击"保存"按钮保存修改，或点击"取消"放弃修改
